rwx (read write execute kao dozvole fajlova na linuksu).
Editor je napisan u C-u i konfiguriše se kao suckless softver uz pomoć patch-eva.

Tutorial koji pratim za osnove projekta: https://viewsourcecode.org/snaptoken/kilo/index.html

Naravno plan je da dodam još mnogo opcija koje ću koristiti za sopstvene potrebe, pritom ću napraviti da se lako može kod modifikovati kako bi postigao princim patch-ovanja kao u suckless softveru. Takodje ću napraviti jedan patch kako bi novi korisnici znali kako da naprave patch-eve kao plugin-ove.

Više o suckless tipu softvera možete naći na https://suckless.org/philosophy/ ili pogledajte ovaj video https://youtu.be/3C6saSpX4KQ?si=YgAMYjCY9coKGpXM
